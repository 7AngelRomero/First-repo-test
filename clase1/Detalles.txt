//Fundamentos de git
git hol amundo/canal de youtube

Branch = Rama de nuestro proyecto, existen ramas Main y copias.
Commit = compromiso, cambio o agregador de características de una rama.
Merge = unir los cambios de la rama copiadas a las ramas main.

Commandos de git:

- git init
- git branch -M main
- git status
